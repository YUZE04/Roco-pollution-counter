# 污染计数器 v1.2.0 — PyQt6 重写版

全新 UI、更多功能、更流畅的动效。

## ✨ 重大变化

### UI 全面重写（PyQt6）
- **自定义深紫标题栏**，不再是原生 Windows 白条
- **Spring 弹簧数字过渡**：今日污染数变化带轻微回弹
- **Tab 横向滑入动画**
- **Card 鼠标悬停紫色描边高亮**
- **17 个矢量图标**（QPainter 绘制，零外部资源）

### 数据可视化
- 统计表每行右侧嵌入**紫 → 青渐变进度条**，按最大值归一化
- "每日污染总数"卡片右上嵌入**最近 7 天迷你折线图**（spark-line）

### Toast 通知系统
- 右下角堆叠式通知，自动 2.4s 淡出
- "+1 恶魔狼"、"已修改"、"已识别" 等瞬时事件自动弹 Toast
- 持久状态（"监测中"/"未启动"）仍留在底部状态条

## 🎯 新功能

### 一键识别游戏窗口
- 设置页点 **识别游戏窗口**
- 自动查找洛克王国窗口 → 读取客户区尺寸 → 套分辨率预设 → 叠加屏幕偏移
- 告别手动填三组坐标的时代

### 统计修改
- **双击**任意统计表行 → 改数量 / 删除
- **添加或修改** 按钮新增精灵条目 / 补录某一天
- 三张表（今日精灵 / 历史累计 / 每日总数）全部支持编辑

### 打赏作者
- 关于页新增 **打赏作者** 按钮
- 弹窗显示支付宝账号并自动复制到剪贴板
- 支持放 `alipay_qr.png` 自动显示收款码

### 悬浮窗升级
- **状态点呼吸动效**（监测中会一闪一闪）
- **右上角「右键菜单」胶囊提示**（未锁定时显示）
- 大数字变化带 300ms OutCubic 平滑过渡
- 顶部玻璃感紫色渐变高光

## 🛠️ 稳定性

- **OCR 引擎无变化**：继续复用 v1.1.x 的 LocalPaddleOCRReader 及所有打补丁
- 所有 paddle/pyclipper 修复、ruamel.yaml/filelock 打包问题全部保留
- 兼容现有 `pollution_config.json` / `pollution_count.json`
