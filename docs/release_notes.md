## v1.1.2 更新内容

- **修复同一场战斗被重复计数**：加入触发沿检测，关键字消失前不再二次计数（解决「连续计数两次」问题）
- **自动识别冷却时间 8 秒 → 12 秒**（手动 +污/-污 热键不受影响）
- **进一步优化鼠标卡顿**：
  - 工作线程优先级降为 `BELOW_NORMAL`，保证 UI 线程能及时处理鼠标消息
  - UI 线程磁盘写入节流 2 秒，避免每次命中都同步写 JSON
  - 降低点击穿透守护频率（700ms → 2500ms）
  - OCR 就绪后停止状态轮询
