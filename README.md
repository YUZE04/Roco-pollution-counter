# Roco Kingdom World Pollution Counter

一个用于洛克王国世界污染追踪的 Python 桌面计数工具。  
A Python desktop counter tool for Roco Kingdom world pollution tracking.

## Features

- OCR text recognition  
  OCR文本识别

- Auto counting  
  自动计数功能

- Manual hotkey add and subtract  
  手动快捷键加减功能

- Compact floating window  
  紧凑型浮动窗口

- Transparency adjustment  
  透明度调整

- Resolution switching  
  分辨率切换功能

- 150% scaling preset support  
  150%缩放预设支持

- Daily statistics  
  每日统计数据

- Species statistics  
  物种统计信息

## Requirements 要求
- Windows
- Python 3.10+
